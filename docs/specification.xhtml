Описание модуля «Файловое досье»

Требования к системе

  1. Сохранение и загрузку файлов в досье

  2. Модель описания структуры досье в формате XML

  3. Формирование печатных форм и иных представлений данных

    1. Представление xml данных в виде pdf с использованием xslt шаблонов и pdfgen/fop

    2. Представление json/xml данных в виде odt/ods с использованием xslt шаблов

  4. Автоматическое сопоставление сканируемых страниц со страницами исходного документа

  5. Проверку наличия подписей в сканированном документе
